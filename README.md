# Usage

<!-- usage -->

```sh-session
$ npm install -g checkly-cli
$ checkly-cli COMMAND
running command...
$ checkly-cli (-v|--version|version)
checkly-cli/0.0.0 darwin-x64 node-v12.18.2
$ checkly-cli --help [COMMAND]
USAGE
  $ checkly-cli COMMAND
...
```
